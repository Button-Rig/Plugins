## Introduction
This repository contains source for plugins for Button Rig, a macro utility desktop application. You can purchase the application at <a href="https://buttonrig.com">ButtonRig.com</a>. If you are a developer, you can reach out to us <a href="https://x.com/ButtonRigApp">@ButtonRigApp</a> on twitter to get a free copy of the application.

<ul>
    <li>For the complete guide on how to develop a plugin, visit <a href="https://docs.buttonrig.com">docs.ButtonRig.com</a>. </li>
    <li>Stuck on something? Feel free to ask for help in the forum at <a href="https://forum.buttonrig.com/topic/plugins">forum.ButtonRig.com/topic/plugins</a>.</li>
    <li>Need us to develop a Button Rig Plugin for your software or business, reach out to us at <a href="mailto:contact@buttonrig.com">contact@buttonrig.com</a>.</li>
</ul>

## Plugin Types
1. **Unsigned Plugins:** 
\
When a user loads an unsigned plugin into Button Rig, the user will see a warning stating that this plugin is unsigned and cannot be trusted.

2. **Signed Plugins:** 
\
The source of signed plugins have been reviewed by us, the plugin is build from source and packaged by us, and have been signed and are available at the <a href="https://forum.buttonrig.com/plugins">Plugin Marketplace</a> for purchase or free download. These are trusted plugins and user will not recieve any warnings while loading this plugin into Button Rig.

3. **Open-Source / Source-Available:** 
\
This repository only contains open-source/source-available plugins. You can raise a pr with your plugin. Once approved, it will be signed and will be available in the <a href="https://forum.buttonrig.com/plugins">Plugin Marketplace</a> for free.

4. **Closed-Source Signed Plugin:** 
\
These are plugins which are closed source, but have been privately reviewed by us, build from source by us, and are signed, and available in the <a href="https://forum.buttonrig.com/plugins">Plugin Marketplace</a>. This is the option if you need to put up your plugin for sale in the plugin marketplace. Contact us at <a href="mailto:contact@buttonrig.com">contact@buttonrig.com</a> to get your closed source plugin into the marketplace.


## How to develop plugins for Button Rig?
Visit <a href="docs.buttonrig.com/plugins">docs.ButtonRig.com</a> for the complete guide to create a plugin. This repository contains open-source/source-available plugins, which you can use as examples for building your plugin.
